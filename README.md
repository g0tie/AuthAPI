<h1>AuthAPI</h1>
<h2>User registering and authorizing API made with Node.js, express, mongoose and jwt</h2>

<p>I made this little project to better understand node.js and how token works.</p>

<code>npm run start</code> to start project
</br>
<code>npm run dev</code> use nodemon for dev

Routes:
<p>Base url : <strong>/api/auth/</strong></p>
<ul>
  <li><strong>/register</strong> : to register a new user</li>
  <li><strong>/login</strong>  : to obtain a jsonwebtoken</li>
  <li><strong>/activate</strong> : to activate an account </li>
  <li><strong>/refresh</strong> : to get a new token based on previous token</li>

</ul>
